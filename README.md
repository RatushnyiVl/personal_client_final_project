## Можливості

### Контакти
- Додавання, редагування та видалення контактів
- Збереження номерів телефону, email, дня народження, адреси
- Відображення контактів з наступаючим днем народження
- Пошук контактів

### Нотатки
- Створення, редагування та видалення нотаток
- Пошук нотаток за текстом
- Пошук та фільтрація нотаток за тегами
- Сортування нотаток за тегами

### Збереження
- Автоматичне збереження даних між сесіями (`.pkl`)
- Експорт контактів та нотаток у формат CSV

##  Встановлення

1. Переконайтесь, що у вас встановлено **Python 3.10+**:
```bash
   python --version
```

2. Клонуйте або завантажте репозиторій:
```bash
   git clone https://github.com/RatushnyiVl/personal_client_final_project.git
```
   Або просто завантажте архів репозиторію

3.  Переконайтеся що ви знаходитеся в директорії проєкту `personal_client_final_project`

4. Встановіть пакет:
```bash
   pip install -e .
```

4. Запустіть з будь-якого місця системи:
```bash
   assistant
```

Після запуску ви побачите вітання:
```
Вітаємо у персональному помічнику!
Введіть команду:
```

## Використання

Введіть команду у консолі та натисніть Enter. Для завершення роботи введіть:

```
вихід
```
або
```
закрити
```

### Приклади команд

| Дія | Команда |
|-----|---------|
| Додати контакт | `додати-контакт` |
| Знайти контакт | `знайти-контакт <ім'я>` |
| Видалити контакт | `видалити-контакт <ім'я>` |
| Додати нотатку | `додати-нотатку <текст>`|
| Знайти нотатку | `знайти-нотатку <текст>` |
| Знайти за тегом | `знайти-тег <тег>` |
| Показати найближчі дні народження | `дні-народження <кількість_днів>` |
| Вийти | `вихід` |

> **Примітка:** Повний перелік команд залежить від реалізації `CommandHandler`. Введіть `допомога` для отримання списку доступних команд.

## Зберігання даних

Дані зберігаються автоматично у домашній директорії користувача:

```
~/.personal_assistant/
├── assistant_data.pkl   # Основний файл збереження (бінарний)
├── contacts.csv         # Експорт контактів
└── notes.csv            # Експорт нотаток
```

- **`assistant_data.pkl`** — основне сховище, завантажується при кожному запуску.
- **`contacts.csv` / `notes.csv`** — зручний формат для перегляду або імпорту в інші програми.

---

## Обробка помилок

Усі команди захищені декоратором `@input_error`, який перехоплює:
- Помилки валідації (наприклад, невірний формат телефону, дня народження, email)
- Відсутність контакту або нотатки
- Неповні аргументи команди

Програма не завершується аварійно — помилка виводиться як повідомлення у консолі.

## Валідація

- **Номер телефону** — має містити рівно **10 цифр** (без пробілів, дефісів чи інших символів).  
  Приклад: `0671234567`

- **Email** — має відповідати стандартному формату `користувач@домен.розширення`.  
  Приклад: `user@example.com`

-  **День народження** — має відповідати формату: `ДД.ММ.РРРР`.  
 Приклад: `21.01.1992`
